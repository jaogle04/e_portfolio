Projects

Name: MATLAB Robot

Role: Project Manager

The focus of this project was to design a code using MATLAB that would give directions to the robot. My team designed a remote to control the robot while also displaying data on the laptop such as speed and a video feed. The speed of the robot would be set through an input on the screen and the directions of the robot were controlled by the arrow keys on the keyboard. We then presented the code and what would be the purpose of the robot in a real-world scenerio and how it would be used.

Name: Circuit's Project

This project used the knowledge of circuits that we had learned recently over the course of a module in EF151.
